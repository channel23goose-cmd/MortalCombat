# Mortal Kombat: Maximum Realism

A premium, single-file fighting game prototype built with HTML5 Canvas.

## 🕹️ Gameplay Features

- **Procedural Animations**: Characters are built from multiple body parts with smooth breathing and swaying effects.
- **Cinematic Feel**: 
  - **Screen Shake**: Intense feedback on big hits.
  - **Hit-Stop**: Weighty impact with brief freeze-frames.
  - **Particle System**: Dynamic blood/spark sprays on hit.
- **Dynamic Arena**: 3-layer parallax background (mountains, ruins, foreground) that shifts as you move.
- **Premium UI**: Gothic MK-style aesthetics, glowing health bars, and a "FINISH HIM!" game-over sequence.

## ⌨️ Controls

### Player 1 (Blue/Azure)
- **W**: Jump
- **A**: Move Left
- **D**: Move Right
- **Space**: Attack

### Player 2 (Red/Crimson)
- **Up Arrow**: Jump
- **Left Arrow**: Move Left
- **Right Arrow**: Move Right
- **Enter**: Attack

## 🛠️ Built With
- HTML5 Canvas
- Vanilla JavaScript
- Google Fonts (Cinzel Decorative, Metal Mania)

## 📜 How to Run
Simply open `index.html` in any modern web browser.
